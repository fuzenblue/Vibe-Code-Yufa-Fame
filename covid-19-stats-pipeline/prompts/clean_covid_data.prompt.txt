You are an Apache Airflow expert. Write a Python function called clean_covid_data(**context) for a PythonOperator.

The function should:
1. Pull raw CSV path from XCom
2. Load only these columns: iso_code, location, date, new_cases, new_deaths, total_cases, total_deaths, population
3. Apply cleaning:
- Filter out OWID aggregate rows (iso_code starts with "OWID_")
- Parse 'date' column as datetime
- Replace negative new_cases or new_deaths values with 0 (corrections in source data)
- Fill null new_cases and new_deaths with 0
- Remove rows where population is null or <= 0
- Remove duplicate (iso_code, date) pairs
4. Log row counts after each step
5. Save to /tmp/covid_clean.csv and push path to XCom
